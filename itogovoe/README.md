# Описание работы скрипта

Скрипт `main.py` — для сбора, анализа и визуализации данных об угрозах информационной безопасности.

### Что делает скрипт

**Этап 1 — Сбор данных** (два источника):
- **Логи Suricata** (`eve.json`) — файл с событиями сетевой безопасности, источник: [suricata-sample-data](https://github.com/FrankHassanabad/suricata-sample-data).
- **API Vulners** — запрос к публичному API для поиска известных уязвимостей.

**Этап 2 — Анализ данных:**
- Парсинг логов и подсчёт статистики: типы событий, топ IP-адресов по числу алертов, частые DNS-запросы.
- Анализ уязвимостей из API: извлечение CVSS, выявление критических уязвимостей.

**Этап 3 — Реагирование на угрозы:**
- Имитация блокировки IP-адресов, которые встречаются в алертах ≥ 5 раз.
- Имитация отправки уведомлений.

**Этап 4 — Отчёт и визуализация:**
- Сохранение результатов в файл `report.csv`.
- Построение графика `chart.png` с двумя диаграммами: топ-10 IP по алертам и распределение CVSS.

### Структура проекта

```
├── main.py        — основной скрипт
├── release/eve.json       — логи Suricata
├── report.csv     — отчёт (генерируется скриптом)
├── chart.png      — график (генерируется скриптом)
└── README.md      — описание проекта
```

### Установка зависимостей

```bash
pip install requests pandas matplotlib
```

### Запуск

```bash
python main.py
```

### Используемые библиотеки

- `requests` — HTTP-запросы к API Vulners
- `pandas` — анализ и обработка данных
- `matplotlib` — построение графиков
- `json` — парсинг логов Suricata
